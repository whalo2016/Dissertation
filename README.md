# Dissertation
## Godavari Basin ML Modelling
This repository contains all R scripts used in the dissertation:

Hydrological Heterogeneity and the Performance of Random Forest Machine-Learning Models: A PCA-Clustering approach in the Godavari River Basin

## Contents
- 01_Data_Preprocessing.R
- 02_Baseline_Model.R
- 03_PCA_Clustering.R
- 04_Tailored_Models.R

## Requirements
This project uses the CAMELS-India dataset (Mangukiya et al., 2025).

The dataset can be downloaded from: https://zenodo.org/records/14999580

R version 2025.5.1

Packages: dplyr, tidyr, purrr, lubridate, readr, ranger, hydroGOF, cluster, ggplot2, mclust, factoextra

## How to run
Scripts are numbered in the order they should be executed
